# Ad Report Agent

Multi-Agent Ad Report Analysis System

## 智能洞察功能

本系统内置规则引擎，能够自动识别广告数据中的 **9类问题（P01-P09）** 和 **10类亮点（A01-A10）**，为优化决策提供智能建议。

### 9类问题识别（P01-P09）

| 编码 | 问题名称 | 描述 |
|------|----------|------|
| P01 | 受众定位偏差 | CTR正常但CVR极低，吸引了非目标用户点击 |
| P02 | 创意疲劳衰减 | 连续多日CTR下降，用户产生审美疲劳 |
| P03 | 时段投放浪费 | 某些时段消耗占比高但CPA显著偏高 |
| P04 | 频次失控 | 高频用户消耗大量预算但转化贡献极低 |
| P05 | 流量作弊嫌疑 | CTR异常高同时跳出率极高，疑似虚假流量 |
| P06 | 出价策略异常 | CPC环比暴涨，竞价环境突变 |
| P07 | 地域渗透饱和 | 某地区人群覆盖率高但边际成本上升 |
| P08 | 设备兼容问题 | 某设备端表现显著差于其他设备 |
| P09 | 竞品活动冲击 | CPC暴涨同时CTR下降，典型竞品大促特征 |

### 10类亮点识别（A01-A10）

| 编码 | 亮点名称 | 描述 |
|------|----------|------|
| A01 | CTR表现优异 | CTR超过基准值3倍，点击效率极高 |
| A02 | CVR表现优异 | CVR超过基准值3倍，转化效率突出 |
| A03 | CPC成本优势 | CPC仅为均价50%，获客成本优势明显 |
| A04 | 消耗曲线健康 | 预算利用率>90%，投放节奏平稳 |
| A05 | 频次控制良好 | 平均曝光频次在1.5-2.5次理想区间 |
| A06 | 转化节奏理想 | 24小时内转化占比>80%，用户决策链路短 |
| A07 | 分时段反差亮点 | 某时段CVR是整体的3倍以上 |
| A08 | 分设备反差亮点 | 某设备ROI是其他设备的2倍以上 |
| A09 | 分素材反差亮点 | 低CTR素材有超高CVR，属于精准定向型 |
| A10 | 展示份额优势 | 展示份额>60%，在竞争中占据主导地位 |

### 自动识别机制

系统在每次数据查询后自动触发规则引擎扫描：
1. 基于查询结果和上下文基准值执行所有规则
2. 每个规则独立判断，符合条件即生成对应的洞察
3. 洞察包含置信度、证据、优化建议等结构化信息
4. 单个规则失败不影响整体执行，保证系统稳定性
