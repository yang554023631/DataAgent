# 点击成本 CPC 与 千次展示成本 CPM

## CPC (Cost Per Click) 点击成本

### 定义
CPC 是平均每次点击的费用，是衡量投放效率的重要指标。

**计算公式**：
```
CPC = 消耗 / 点击次数
```

**示例**：消耗1000元，获得500次点击，则 CPC = 1000 / 500 = 2元

### 影响因素
1. **出价**：出价越高，CPC通常越高
2. **CTR**：CTR越高，系统判定素材质量越好，实际CPC越低
3. **行业竞争**：竞争激烈的行业CPC更高
4. **定向范围**：定向越窄，CPC通常越高

### 优化方向
- 提高素材质量，提升CTR，降低实际CPC
- 优化定向，避免与竞争对手正面竞争
- 分时段调整出价，在低峰期降低出价

---

## CPM (Cost Per Mille) 千次展示成本

### 定义
CPM 是每获得1000次曝光的费用，衡量曝光成本。

**计算公式**：
```
CPM = 消耗 / 曝光次数 × 1000
```

**示例**：消耗1000元，获得50000次曝光，则 CPM = 1000 / 50000 × 1000 = 20元

### 影响因素
1. **出价方式**：CPM出价直接影响CPM成本
2. **素材质量**：高质量素材系统会有更低的CPM
3. **人群质量**：高价值人群的CPM通常更高
4. **投放时段**：高峰期CPM更高

### 优化方向
- 避开竞争激烈的高峰时段
- 优化素材质量，获得系统流量倾斜
- 拓展定向范围，寻找低价流量

---

## CPC 与 CPM 的关系

### 公式推导
```
CPC = CPM / (CTR × 10)
```

**解释**：
- CTR = 1% 时，CPC = CPM / 10
- CTR = 2% 时，CPC = CPM / 20
- CTR = 5% 时，CPC = CPM / 50

**关键结论**：**CTR提升一倍，CPC降低一半。

---

## 指标解读误区

### ❌ 误区1：CPC越低越好
- CPC低说明点击成本低，但不代表最终效果好
- 如果CPC很低但转化率也低，最终转化成本可能反而更高
- 需要结合转化率和转化成本综合判断

### ❌ 误区2：CPM高说明投放差
- CPM高可能是因为竞争激烈或人群质量高
- 如果CTR和转化率都高的话，最终转化成本可能更低
- 不能只看CPM绝对值，要看最终的转化成本和ROI

### ❌ 误区3：CPC和CPM不能同时优化
- 实际上，提升素材质量（CTR）可以同时降低CPC和CPM
- 优质素材会获得系统的流量倾斜和成本优势

## 优化策略矩阵

| 场景 | 策略 |
|------|------|
| CPC高，CPM正常 | 优化素材，提升CTR；降低出价 |
| CPC正常，CPM高 | 拓展定向，寻找低价流量；避开高峰 |
| CPC和CPM都高 | 检查是否定向过窄；考虑换出价方式；检查行业竞争是否过于激烈 |
| CPC和CPM都低 | 这是好现象！加大预算投放，抓住红利期 |
