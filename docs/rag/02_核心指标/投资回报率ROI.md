# 投资回报率 (ROI - Return on Investment)

## 定义
投资回报率是衡量广告投放效益的核心指标，表示每投入一元广告费用带来的收入回报。

## 计算公式
```
ROI = (广告带来的总收入 - 广告总花费) / 广告总花费 × 100%
```

简化公式：
```
ROI = (总收入 / 总花费 - 1) × 100%
```

例如：
- 广告总花费：10,000 元
- 广告带来的总收入：30,000 元
- ROI = (30,000 - 10,000) / 10,000 × 100% = 200%

## 指标意义
1. **最终效果**：ROI 是评估广告效果的最终指标，直接关联盈利
2. **投放决策**：ROI 决定是否继续投放、增加或减少预算
3. **渠道对比**：不同渠道的 ROI 对比可指导预算分配

## 关键理解
- **ROI > 0**：广告投放盈利，值得继续投入
- **ROI = 0**：广告投放盈亏平衡
- **ROI < 0**：广告投放亏损，需要优化或停止

## 优化策略
1. **提升转化价值**：通过捆绑销售、 upsell 提高客单价
2. **降低获客成本**：优化 CPA，降低每转化成本
3. **提升转化率**：优化落地页、转化路径
4. **精准用户分层**：针对高价值用户提高出价和预算
5. **归因优化**：准确归因，合理分配预算到各触点
