# 转化成本 (CPA - Cost Per Acquisition)

## 定义
转化成本是指获得每一个转化（如用户注册、购买、下载等）所花费的广告费用。它是衡量广告投入产出效率的关键指标。

## 计算公式
```
CPA = 广告总花费 / 转化次数
```

例如：
- 广告总花费：10,000 元
- 转化次数：50 次
- CPA = 10,000 / 50 = 200 元/次

## 指标意义
1. **获客效率**：CPA 直接反映了获取每个客户的成本
2. **盈利红线**：CPA 必须低于用户生命周期价值 (LTV) 才能盈利
3. **渠道质量**：不同渠道的 CPA 对比可以评估渠道质量

## 行业参考值
- 电商行业：50 - 200 元
- 游戏行业：20 - 100 元
- 教育行业：100 - 500 元
- 金融行业：200 - 1,000 元

## 优化方向
1. **提高转化率**：优化落地页、简化转化流程
2. **精准定向**：缩小定向范围，只触达高转化潜力用户
3. **素材优化**：突出产品核心价值，吸引高意向用户
4. **出价策略**：根据转化数据动态调整出价
5. **负面过滤**：及时暂停低转化高消耗的广告
