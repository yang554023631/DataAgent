# 预算分配与出价策略

预算和出价直接决定了广告的曝光量级和最终成本。合理的预算分配和出价策略可以在同样预算下获得更好的效果。

## 预算分配原则

### 1. ROI导向原则
**核心**：谁的ROI高，就给谁更多预算。

**操作步骤**：
1. 计算各计划/广告组的ROI
2. ROI高的加大预算
3. ROI低的缩减或暂停预算
4. 定期（每天/每周）重新评估调整

### 2. 分散风险原则
不要把所有预算集中在1-2个计划上：
- 主力计划：60%-70%预算，稳投
- 测试计划：20%-30%预算，探索
- 备用计划：10%预算，应急

### 3. 生命周期原则
不同阶段的计划分配不同预算：
| 阶段 | 预算占比 | 说明 |
|------|---------|------|
| 测试期 | 10%-20% | 小预算测试，找到优质素材和定向 |
| 放量期 | 50%-70% | 对测试通过的计划加大预算 |
| 稳定期 | 20%-30% | 成熟计划稳定投放 |
| 衰退期 | < 10% | 逐步减少，准备新计划接替 |

---

## 预算分配方法

### 方法1：等预算测试法
**场景**：多个新计划同时启动

**操作**：
- 每个计划分配相同的测试预算（如每天500元）
- 测试3-7天
- 关停ROI不达标的计划
- 把预算集中到效果好的计划上

**优点**：公平测试，不漏掉潜力计划
**缺点**：预算分散，好计划可能前期量不够

### 方法2：梯度放量法
**场景**：找到优质计划后逐步加大预算

**操作**：
- 第一天：500元 → 效果好
- 第二天：1000元 → 效果稳定
- 第三天：2000元 → 继续观察
- 以此类推，每次加量不超过前一天的100%

**优点**：风险可控，避免预算浪费
**缺点**：放量速度可能不够快

### 方法3：动态预算分配
**场景**：成熟期账户，计划较多

**操作**：
- 每24小时评估一次各计划的ROI
- 自动把预算从低ROI计划转移到高ROI计划
- 保持总预算不变的情况下提升整体效果

## 出价策略基础

### 出价的核心逻辑
```
实际成本 = 目标出价 × (实际CTR / 预期CTR) × (实际CVR / 预期CVR)
```

系统会根据实际转化效果自动调整实际扣费，不是简单按出价扣费。

### 出价与量的关系
- 出价越高 → 竞争力越强 → 曝光越多
- 出价越低 → 竞争力越弱 → 曝光越少

但出价与成本**不是线性关系**：
- 出价从10元涨到20元 → 曝光可能涨200%，成本可能只涨20%
- 出价从50元涨到100元 → 曝光可能只涨30%，成本可能涨80%

---

## 出价调整时机与方法

### 什么时候该调价？

✅ **应该调价的情况**：
1. 成本持续高于目标20%以上，且连续3天
2. 量太小，预算花不出去，需要提价拿量
3. 成本远低于目标，可以提价扩量
4. 节假日或活动期，竞争环境变化

❌ **不应该调价的情况**：
1. 当天数据还没稳定（至少看半天数据）
2. 样本量太小（转化数<10）
3. 频繁调整（一天调好几次）
4. 同时调整出价、定向、素材多个变量

### 调价幅度建议

| 场景 | 建议幅度 | 说明 |
|------|---------|------|
| 成本高很多 | -10% ~ -20% | 不要一次降太多 |
| 成本略高 | -5% ~ -10% | 小步调整 |
| 成本达标想扩量 | +5% ~ +10% | 稳步提价 |
| 成本很低量大 | +10% ~ +20% | 可以大胆提 |

**原则**：每次调整幅度不超过20%，每天调整不超过1次。

### 调价后观察周期
- 至少观察24小时再评估效果
- 不要调完几小时看数据不好又调回去
- 给系统足够的学习时间

## 分阶段出价策略

### 冷启动期出价
**目标**：快速积累数据，跑出第一波转化

**策略**：
- 出价略高于目标成本（高10%-20%）
- 先跑出数据再说，成本可以后面优化
- 不要一开始就卡很死的成本，数据最重要

**注意**：设置好预算上限，控制风险

### 优化期出价
**目标**：在保证量级的前提下，逐步降低成本

**策略**：
- 数据积累足够后（建议至少20个转化）
- 成本高就逐步降价，每次降5%-10%
- 观察量级变化，不要降价太猛导致量掉光
- 找到"成本-量级"的最佳平衡点

### 稳定期出价
**目标**：维持成本稳定，同时尽可能扩量

**策略**：
- 成本稳定后，可以尝试小幅度提价扩量
- 提价5%，观察成本和量级变化
- 如果成本上涨在可接受范围内，就保持
- 如果成本涨太多，就调回原价

### 衰退期出价
**目标**：延长生命周期，平稳过渡

**策略**：
- 效果开始衰退时，不要盲目提价保量
- 可以适当降价，降低成本要求
- 同时准备新计划接替
- 预算逐步转移到新计划

## 分时出价策略

### 为什么要分时出价？
不同时段的竞争环境和用户质量差异很大：
- 高峰期（中午12-14点，晚上20-23点）：竞争激烈，CPM高
- 低峰期（凌晨、早上）：竞争小，CPM低
- 工作日和周末的用户行为也不一样

### 分时出价设置方法

**步骤1：分析时段数据**
- 拉取过去7天分小时的转化成本数据
- 计算每个小时的平均成本
- 找出高成本时段和低成本时段

**步骤2：设置分时出价**
| 时段类型 | 建议出价 | 示例 |
|---------|---------|------|
| 高成本时段 | 目标出价 × 0.8 | 成本比均值高30%以上 |
| 正常时段 | 目标出价 × 1.0 | 成本接近均值 |
| 低成本时段 | 目标出价 × 1.2 | 成本比均值低30%以上 |

**步骤3：定期优化**
- 每周看一次分时数据
- 根据最新数据调整分时出价系数

## 分渠道出价策略

### 不同渠道的特性
| 渠道 | 用户质量 | 竞争程度 | 建议出价策略 |
|------|---------|----------|-------------|
| 信息流 | 中等 | 高 | 稳健出价，注重素材质量 |
| 搜索广告 | 高 | 很高 | 可以出高价，精准关键词 |
| 短视频 | 中等偏高 | 高 | 出价略高，重点看CTR |
| 联盟广告 | 偏低 | 低 | 低价策略，量大从优 |

### 渠道预算分配
根据各渠道的ROI动态调整：
- ROI高的渠道加大预算占比
- ROI低的渠道缩减预算
- 新渠道先小预算测试

## 常见出价误区

### ❌ 误区1：出价越低，成本越低
- 出价太低 → 拿不到曝光 → 没有数据 → 系统无法优化 → 成本反而更高
- 适当的出价才能获得足够曝光，让模型收敛

### ❌ 误区2：出价越高，成本越高
- 系统是按实际效果扣费的，不是按出价
- 出价高可以拿到更好的流量，CTR和CVR可能更高
- 实际成本可能反而更低（出价20元实际成本15元 vs 出价15元实际成本18元）

### ❌ 误区3：频繁调价
- 每次调价系统都要重新学习
- 频繁调价会让模型不稳定
- 建议每天调价不超过1次，每次间隔至少24小时

### ❌ 误区4：所有计划用一样的出价
- 不同计划的素材、定向、阶段都不一样
- 应该每个计划单独设置出价
- 好素材可以高出价，差素材应该低价或直接停

## 预算与出价联动优化

### 经典优化流程

```
第1天：多个小预算计划测试出价
   ↓
第3天：关停效果差的，集中预算给好的
   ↓
第5天：对优质计划逐步提价扩量
   ↓
第7天：根据时段数据设置分时出价
   ↓
持续：每天观察数据，动态调整
```

### 预算用尽怎么办？

**情况1：成本达标，预算很早就花完了**
- ✅ 好事！说明效果很好
- 应该增加预算，同时适当提价
- 抓住机会尽可能扩量

**情况2：成本很高，预算很快花完了**
- ❌ 坏事！说明出价太高或定向太松
- 应该降价，同时收紧定向
- 必要时暂停，重新调整后再投

---

## 实战总结

1. **预算向ROI高的地方倾斜**
   - 不要凭感觉分配，用数据说话
   - 定期review，动态调整

2. **出价要循序渐进**
   - 小步快跑，不要大起大落
   - 每次调整后给系统学习时间

3. **追求整体最优，不是单个计划最优**
   - 单个计划成本高一点，但整体量级和ROI更好也是可以接受的
   - 站在账户全局角度看问题

4. **建立规则，避免情绪化操作**
   - 制定明确的调价规则（如成本高20%才调）
   - 不要看一天数据不好就慌，至少看3天趋势
