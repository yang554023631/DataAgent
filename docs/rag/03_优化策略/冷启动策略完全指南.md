# 冷启动策略完全指南

冷启动是广告投放最关键的阶段，决定了账户后续的效果和天花板。好的冷启动可以让后续投放事半功倍。

## 什么是冷启动？

### 定义
新账户/新计划刚开始投放时，系统没有足够的转化数据，无法准确预测哪些用户会转化，需要一个"学习"的过程，这个过程就是冷启动。

### 冷启动的标志
- 计划刚开始投放的前0-3天
- 转化数积累到20-50个之前
- 系统模型还未稳定，成本波动较大

### 冷启动成功的标准
- 连续3天成本达标
- 量级稳定，预算能够正常花完
- 模型收敛，不需要频繁调整

---

## 冷启动前的准备工作

### 1. 素材准备

**数量要求**：
- 至少准备5-10条不同类型的素材
- 涵盖不同风格、不同角度
- 最好有1-2条历史验证过的爆款素材

**质量要求**：
- 清晰度达标，无拉伸、无黑边
- 文案通顺，无错别字
- 符合平台审核规范，避免违规

### 2. 定向准备

**基本原则**：
- 不要过窄：至少覆盖1000万以上人群
- 不要过宽：避免完全通投（除非有爆款素材加持）
- 核心维度：年龄、性别、地域要准确

**推荐定向组合**：
```
基础属性（年龄+性别+地域） + 3-5个核心兴趣标签
```

### 3. 出价准备

**参考因素**：
- 行业平均出价
- 历史投放数据
- 可接受的目标成本

**出价建议**：
- 比目标成本高10%-20%
- 先拿到数据，成本后面再优化
- 设置好预算上限控制风险

### 4. 预算准备

**每日预算建议**：
```
日预算 = 目标成本 × 20 ~ 目标成本 × 50
```

**示例**：目标成本50元 → 日预算1000-2500元

**原因**：
- 至少要让系统拿到20个以上转化才能学习
- 预算太少，数据积累太慢，冷启动时间拉长

## 冷启动期的操作策略

### Day 0-1：投放首日

**目标**：让计划顺利跑起来，拿到第一批数据。

**重点观察**：
- 计划是否正常消耗
- CTR是否在正常范围（>1%）
- 有没有转化进来

**禁止操作**：
❌ 不要几个小时就调整
❌ 不要因为没有转化就立刻停掉
❌ 不要频繁改出价和定向

### Day 2-3：观察与微调

**目标**：观察数据趋势，做少量必要调整。

**调整策略**：

| 情况 | 操作 |
|------|------|
| 消耗正常，有转化，成本略高 | 不动，继续观察 |
| 消耗正常，无转化，CTR正常 | 再给一天时间，耐心等待 |
| 消耗正常，无转化，CTR很低 | 考虑换素材 |
| 消耗很少，跑不动 | 提价10%-20%，或放宽定向 |
| 跑得很快，成本远高于目标 | 降价10%，或收紧定向 |

### Day 4-7：数据验证与决策

**目标**：判断冷启动是否成功，决定后续策略。

**决策矩阵**：

| 转化数 | 成本情况 | 决策 |
|--------|---------|------|
| ≥ 20个 | 达标或接近达标 | ✅ 冷启动成功，进入优化期 |
| ≥ 20个 | 远高于目标 | ⚠️ 逐步降价优化，或暂停 |
| < 10个 | 无转化或很少 | ❌ 果断暂停，换方案重投 |

## 提升冷启动成功率的技巧

### 技巧1：用好"老带新"

**方法**：
- 用已经跑起来的老计划，带新计划
- 新计划用老计划的优质素材
- 新计划用老计划验证过的定向

**原理**：
- 优质素材和定向的成功率远高于盲测
- 站在已有成功经验的基础上，成功率提升50%以上

### 技巧2：相似人群冷启动

**方法**：
- 用历史转化用户做种子人群
- 拓展1%-5%的相似人群
- 专门建计划投放这个人群包

**效果**：
- 冷启动成功率提升30%-100%
- 前期成本更稳定，波动更小

**注意**：
- 种子人群质量要高（建议用付费用户，不要只用点击用户）
- 种子人群量级要足够（建议至少1000人以上）

### 技巧3：阶梯出价法

**方法**：
- 第一天高出价（目标×1.2）
- 第二天正常出价（目标×1.0）
- 第三天低成本测试（目标×0.8）
- 看哪个价位效果最好

**适用场景**：
- 完全没有历史数据的新品
- 不确定出价应该设多少

### 技巧4：多计划赛马

**方法**：
- 同时建5-10个计划测试
- 每个计划用不同的素材或定向
- 3天后保留效果好的，停掉效果差的

**优点**：
- 提高命中爆款的概率
- 缩短冷启动时间
- 分散风险，不会全军覆没

**注意**：
- 计划之间不要完全一样，要有差异
- 不要建太多，管理不过来，5-10个合适

## 冷启动期常见问题

### 问题1：跑不动（消耗很慢）

**原因分析**：
1. 出价太低，竞争力不足
2. 定向太窄，覆盖人群太少
3. 素材CTR太低，系统不给量

**解决方案**：
1. 提价10%-20%，观察2小时
2. 放宽定向限制，扩大人群覆盖
3. 更换CTR更高的素材

### 问题2：跑得太快，成本很高

**原因分析**：
1. 出价太高
2. 定向太宽
3. 素材质量差，CVR低

**解决方案**：
1. 降价10%，不要一下降太多
2. 收紧定向，排除低质量人群
3. 优化素材，提升转化率

**注意**：不要直接暂停，给系统一些学习时间。

### 问题3：有点击但没转化

**原因分析**：
1. 素材吸引了非目标用户
2. 落地页承接不好
3. 产品本身吸引力不足

**解决方案**：
1. 检查素材是否过度夸张，实事求是
2. 优化落地页加载速度和内容
3. 优化定向，更精准地触达目标用户

### 问题4：成本波动很大

**正常现象**：
- 冷启动期成本波动大是正常的
- 数据量小，偶然因素影响大

**应对方法**：
- 看3天累计成本，不要看单日
- 转化数<20个时，成本参考价值有限
- 只要整体趋势向好就不用慌

## 冷启动失败的常见原因

### ❌ 失败原因1：预算不足
- 每天预算只够几个转化
- 数据积累太慢，系统学不到东西
- 冷启动时间无限拉长

**解决**：冷启动期预算要充足，保证每天至少能有10-20个转化。

### ❌ 失败原因2：频繁调整
- 一小时看一次数据，不好就调
- 一天改好几次出价和定向
- 系统模型一直被打断，无法收敛

**解决**：每天最多调整1次，每次调整后观察24小时。

### ❌ 失败原因3：素材不行
- 只准备了1-2条素材
- 素材质量差，CTR很低
- 没有备选方案

**解决**：冷启动前准备5-10条素材，多方向测试。

### ❌ 失败原因4：期望过高
- 第一天就要求成本达标
- 一看到成本高就立刻停掉
- 不给系统学习的机会

**解决**：接受冷启动期成本高一些，看3天累计数据。

---

## 冷启动成功后的过渡

### 第一步：稳定期（7-14天）
- 不要立刻大幅加预算
- 保持现有设置，观察数据稳定性
- 确认连续7天成本达标

### 第二步：逐步放量（14天以后）
- 每次加预算不超过前一天的50%
- 加预算的同时可以适当提价5%-10%
- 密切关注成本变化，及时回调

### 第三步：拓展新方向
- 主力计划稳定后，测试新的素材方向
- 测试新的定向组合
- 建立第二、第三增长曲线

---

## 避坑指南总结

1. **耐心比什么都重要**
   - 冷启动需要时间，不要太心急
   - 至少看3天数据再做判断

2. **预算要给足**
   - 舍不得孩子套不着狼
   - 预算太少冷启动很难成功

3. **素材是核心**
   - 好的素材冷启动成功率高很多
   - 多准备素材，多方向测试

4. **不要频繁调整**
   - 系统学习需要时间
   - 频繁调整只会让模型更不稳定

5. **接受不完美**
   - 冷启动期成本高一点很正常
   - 只要趋势向好，后面可以优化
