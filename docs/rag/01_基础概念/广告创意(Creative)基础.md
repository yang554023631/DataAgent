# 广告创意 (Creative) 基础

广告创意是展示给用户看的最终广告内容，是广告投放体系中最末端也是最重要的一环。

## 创意的构成要素

### 1. 素材类型
- **图片素材**：单图、多图、组图
- **视频素材**：横版视频（16:9）、竖版视频（9:16）、方形视频（1:1）
- **纯文案**：信息流文字链广告

### 2. 文案内容
- **标题**：吸引用户点击的核心文案（一般15-30字）
- **描述**：补充说明产品或活动的详细信息（一般30-100字）
- **行动号召 (CTA)**：引导用户点击的按钮文案，如"立即下载"、"了解更多"、"免费试用"

### 3. 落地页
用户点击广告后跳转的页面，直接影响转化率。

## 创意审核规范

### 必须遵守的规则
1. **内容合规**：不得包含虚假宣传、夸大承诺
2. **素材清晰**：图片/视频不得模糊、拉伸
3. **文案通顺**：不得有错别字、语病
4. **品牌一致**：创意内容需与品牌定位一致

### 常见拒审原因
- 包含"最"、"第一"等绝对化用语
- 素材涉及版权问题（如未授权使用明星肖像）
- 落地页无法正常打开
- 涉及敏感行业或内容

## 创意生命周期

### 1. 审核期
- 状态：审核中 / 审核通过 / 审核拒绝
- 时长：一般1-2小时，高峰期可能延长

### 2. 探索期
- 特点：系统探索人群模型，波动较大
- 建议：不要频繁调整，给系统足够的探索时间

### 3. 放量期
- 特点：模型稳定，消耗和转化稳步上升
- 建议：保持关注，可适当提价放量

### 4. 衰退期
- 特点：CTR和转化率持续下降，消耗减少
- 建议：及时上新素材替换，避免账户整体衰退

## 创意优化方向

1. **前3秒抓眼球**：视频前3秒是关键，直接决定用户是否继续观看
2. **痛点前置**：开头直接点出用户痛点，引发共鸣
3. **真实感**：过于"广告感"的素材容易引起反感，真实场景更有代入感
4. **CTA明确**：明确告诉用户下一步该做什么
