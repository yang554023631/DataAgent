# 曝光量 (Impressions)

## 定义
曝光量是指广告在用户设备上被展示的次数。它是广告投放的基础指标，代表广告触达用户的规模。

## 计算方式
曝光量直接统计广告被加载并展示的次数，每次展示计为一次曝光。

## 指标意义
1. **触达规模**：曝光量反映了广告覆盖的用户数量规模
2. **投放广度**：高曝光量说明广告投放覆盖范围广
3. **预算消耗**：曝光量与预算消耗直接相关，CPM 计费模式下曝光越多花费越高

## 影响因素
1. **预算**：预算越高，可获得的曝光量越大
2. **出价**：出价越高，在竞价中越有竞争力，获得更多曝光
3. **定向范围**：定向范围越广，潜在用户池越大，曝光机会越多
4. **素材质量**：高质量素材更容易获得平台推荐，曝光量更高

## 优化建议
1. 适当放宽定向条件，扩大用户池
2. 提高出价，增强竞价竞争力
3. 优化素材质量，提升平台推荐权重
4. 增加投放时段，覆盖更多用户活跃时段
