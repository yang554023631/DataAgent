# 广告主 (Advertiser)

广告主是广告投放的主体，指在广告平台上投放广告的企业或个人。

## 定义

广告主是整个广告投放体系中的最高层级组织单位，所有的广告计划、广告组、广告创意都隶属于某个广告主。

## 层级结构

在主流广告平台（如巨量引擎、百度营销、腾讯广告）中，结构通常为：

```
广告主 (Advertiser)
└── 广告计划 (Campaign)
    └── 广告组 (Ad Group)
        └── 广告创意 (Creative)
```

## 广告主属性

- **广告主 ID**：系统内唯一标识符
- **广告主名称**：企业或品牌名称
- **行业类型**：所属行业分类（如电商、游戏、教育等）
- **账户状态**：正常、审核中、冻结等
- **余额/预算**：账户可用资金

## 常见操作

1. **创建广告计划**：在广告主下新建广告投放计划
2. **设置账户预算**：控制整个广告主的日消耗或总消耗
3. **查看汇总数据**：查看所有广告投放的汇总指标
4. **管理授权**：为子账号或代理商设置操作权限

## 注意事项

- 一个企业可能有多个广告主账户（按产品线、渠道等划分）
- 广告主的资质认证会影响可投放的广告类型和流量
- 账户余额不足时，所有下属广告都会停止投放
