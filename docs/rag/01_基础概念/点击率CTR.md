# 点击率 (CTR - Click-Through Rate)

## 定义
点击率（CTR）是衡量广告效果的核心指标之一，表示广告被点击的次数占广告曝光次数的比例。

## 计算公式
```
CTR = 点击次数 / 曝光次数 × 100%
```

例如：
- 曝光次数：10,000 次
- 点击次数：200 次
- CTR = 200 / 10,000 × 100% = 2%

## 指标意义
1. **广告吸引力**：CTR 直接反映了广告素材、文案对目标用户的吸引力程度
2. **投放精准度**：CTR 高通常说明广告定向准确，触达到了目标用户
3. **平台权重**：在广告竞价系统中，CTR 是影响广告质量得分的重要因素

## 行业参考值
- 搜索广告：2% - 5%
- 信息流广告：0.5% - 2%
- 展示广告：0.1% - 0.5%

## 优化方向
1. 优化广告素材和文案，提高吸引力
2. 优化定向设置，精准触达目标用户
3. 测试不同创意形式，找出最佳组合
4. 优化投放时段，选择用户活跃高峰时段
