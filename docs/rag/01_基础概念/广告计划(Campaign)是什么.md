# 广告计划 (Campaign)

广告计划是广告投放的一级组织单位，用于管理一组相关的广告活动。

## 定义

广告计划是广告主下的第一层投放结构，通常按营销目标、产品线、推广周期等维度划分。

## 主要作用

1. **目标管理**：统一设置投放目标、预算、出价策略
2. **数据汇总**：汇总下属所有广告组的数据表现
3. **批量操作**：支持批量启停、调整预算等操作

## 广告计划常用配置

### 1. 营销目标

- **品牌曝光**：提升品牌知名度，按曝光量计费
- **点击引流**：获取网站访问量，按点击量计费
- **转化成交**：获取用户转化（下单、注册、咨询等）
- **应用推广**：推广 App 下载和激活

### 2. 预算设置

- **日预算**：每日最高消耗金额
- **总预算**：整个投放周期的总消耗上限
- **预算分配方式**：均匀投放 / 优先投放

### 3. 投放时段

- 设置广告投放的具体时间段
- 可按小时精细化设置投放系数

### 4. 投放地域

- 选择投放的省份/城市
- 支持排除特定地区

## 命名建议

建议使用清晰的命名规范，便于后期管理：

```
{营销目标}_{产品线}_{投放渠道}_{日期}
示例：转化_女装_信息流_202401
```

## 常见优化策略

1. 对高转化的计划增加预算
2. 对低 ROI 的计划及时关停
3. 按时段效果调整投放系数
4. 定期测试不同出价策略的效果
