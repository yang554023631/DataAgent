# 广告组 (Ad Group) 详解

广告组是广告投放体系中的二级组织单位，位于广告计划之下、广告创意之上。

## 层级关系

```
广告计划 (Campaign)
└── 广告组 (Ad Group) ← 本层级
    └── 广告创意 (Creative)
```

## 广告组的作用

1. **统一管理**：一个广告组下的创意共享相同的定向设置、出价策略和投放时段
2. **A/B测试**：同一广告组下可以放多个创意，系统自动进行流量分配和效果对比
3. **预算控制**：可以单独设置广告组的日预算或总预算

## 广告组核心设置

### 1. 定向设置
- 人群定向：年龄、性别、兴趣标签等
- 地域定向：省市、商圈等
- 设备定向：iOS、Android、PC等
- 网络环境：WiFi、4G、5G等

### 2. 出价设置
- 出价方式：CPC、CPM、OCPC、OCPM等
- 出价金额：具体的出价数值
- 出价策略：优先跑量、稳定成本等

### 3. 投放设置
- 投放时段：按小时设置投放时段
- 投放速度：匀速投放、尽快投放
- 投放日期：开始日期和结束日期

## 创建广告组的最佳实践

1. **按素材类型分组**：视频素材和图片素材分开放在不同广告组
2. **按定向分组**：差异较大的定向（如不同地域）分成不同广告组
3. **控制创意数量**：每个广告组建议放置3-5个创意，过多会导致测试周期过长
4. **预算合理分配**：重点广告组分配更多预算，保证有足够的探索空间

## 常见问题

**Q: 一个广告计划下放多少个广告组合适？**
A: 建议5-10个，根据预算规模和运营人力调整。过少不利于充分探索，过多会增加管理成本。

**Q: 广告组下的创意之间会抢量吗？**
A: 会的，同一广告组下的创意共享定向和流量池，系统会自动分配流量给效果更好的创意。
