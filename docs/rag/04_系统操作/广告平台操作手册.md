# 广告平台操作手册

本文档涵盖了主流广告平台的日常操作流程和注意事项。

---

## 一、日常操作流程

### 每日必做（30分钟）

| 时间 | 操作内容 | 说明 |
|------|---------|------|
| 上午9-10点 | **查看昨日数据** | 消耗、转化、成本、ROI是否正常 |
| 上午10-11点 | **账户巡检** | 有没有计划超成本、素材审核不通过、余额不足 |
| 下午3-4点 | **实时数据监控** | 看当日消耗速度、转化趋势 |
| 下午5-6点 | **当日调整** | 根据数据做必要的出价、预算调整 |

### 查看数据要点

1. **先看整体**：账户总消耗、总转化、平均成本
2. **再看计划**：哪些计划花了钱，效果怎么样
3. **再看素材**：哪些素材在跑，CTR怎么样
4. **最后看细分**：分时段、分地域、分定向维度分析

### 账户巡检清单

- [ ] 账户余额充足（至少够3天消耗）
- [ ] 所有计划状态正常（没有被误暂停）
- [ ] 所有素材审核通过（没有审核不通过）
- [ ] 没有超成本严重的计划
- [ ] 没有计划预算花完但效果还很好的

## 二、新建计划操作规范

### 创建前准备

✅ **确认素材准备好**：至少3-5条素材
✅ **确认定向策略**：明确要投什么人群
✅ **确认出价策略**：参考历史数据和行业均值
✅ **确认预算设置**：冷启动期预算要充足

### 创建步骤

**Step 1：基础设置**
```
计划名称：[日期]_[产品]_[定向类型]_[测试编号]
   示例：20240115_游戏A_兴趣标签_01
   好处：一看就知道是什么计划，方便后面分析
投放日期：从今天开始，不设结束日期（方便手动控制）
投放时段：初期全时段投放，有数据后再设分时
```

**Step 2：定向设置**
```
年龄：只排除明显不相关的，不要卡太死
性别：不限或按产品特性
地域：只排除明显不相关的地区
兴趣：3-5个核心标签，不要选太多
设备：按产品适配情况
```

**Step 3：出价与预算**
```
出价方式：OCPM优先，冷启动效果好
目标出价：比可接受成本高10%-20%
日预算：目标成本 × 30 ~ 目标成本 × 50
        （保证至少能拿到30个转化数据）
```

**Step 4：上传素材**
```
每个计划上传3-5条不同素材
给素材也命名编号，方便追踪效果
不要重复上传一模一样的素材
```

### 创建后检查

- [ ] 计划状态是"投放中"
- [ ] 预算、出价设置正确
- [ ] 素材全部审核通过
- [ ] 定向没有选错

## 三、数据报表操作

### 常用报表维度

| 报表类型 | 用途 | 查看频率 |
|---------|------|---------|
| **分计划报表** | 看各计划效果对比 | 每天 |
| **分素材报表** | 看哪些素材效果好 | 每天 |
| **分时段报表** | 看什么时间效果好 | 每周 |
| **分地域报表** | 看哪些地区效果好 | 每周 |
| **分人群报表** | 看各人群包效果 | 每周 |

### 数据导出与备份

**导出频率**：
- 每日数据：每天导出保存
- 周度汇总：每周一导出上周数据
- 月度汇总：每月初导出上月数据

**保存格式建议**：
```
数据/
  ├── 日报/
  │   └── 202401/
  │       └── 20240115_账户数据.csv
  ├── 周报/
  │   └── 2024年第3周_数据汇总.xlsx
  └── 月报/
      └── 2024年1月_月度报告.xlsx
```

## 四、素材管理操作

### 素材命名规范

```
[日期]_[素材类型]_[卖点]_[编号]
  ↓
示例：
20240115_视频_痛点型_01
20240115_图片_效果对比_03
```

**为什么要规范命名？**
- 看名字就知道素材内容
- 方便统计不同类型素材的成功率
- 方便后续查找和复用

### 素材库管理

**文件夹分类**：
```
素材库/
  ├── 待测试/       新做的素材，还没上线
  ├── 测试中/       正在测试的素材
  ├── 爆款库/       验证过的优质素材
  ├── 备选库/       效果一般但还能用的
  └── 废弃库/       效果差或违规的
```

**定期整理**：
- 每周整理一次素材库
- 及时归档素材，标注素材表现
- 爆款素材单独备份，防止丢失

## 五、账户资金管理

### 充值时机判断

**预警线**：余额 < 3天消耗 → 该充值了
**安全线**：余额 = 7天消耗 → 比较安全
**舒适线**：余额 > 15天消耗 → 不用操心

### 充值操作注意事项

1. **提前充值**：不要等余额快用完才充，到账可能有延迟
2. **走对公流程**：保存好合同、发票，财务合规
3. **索要返点**：大额充值可以跟渠道谈返点
4. **记录到账**：充值后确认到账，记录在台账

### 每日对账

- 平台消耗金额 vs 财务记账金额
- 发现差异及时找渠道核对
- 每月底做一次总账核对

## 六、多账户管理

### 账户命名规范

```
[公司]_[产品]_[投放平台]_[编号]
  ↓
示例：
XX公司_游戏A_抖音_01
XX公司_游戏A_快手_01
```

### 账户权限管理

| 角色 | 权限 | 人数 |
|------|------|------|
| **管理员** | 全部权限 | 1-2人（核心负责人） |
| **优化师** | 操作权限，不能改密码和支付 | 按需要设置 |
| **观察者** | 只读权限，看数据 | 老板、财务等 |

### 账户隔离策略

- **不同产品用不同账户**：避免相互影响
- **测试账户和主力账户分开**：测试不要影响主力投放
- **备用账户常备**：主账户出问题时能立刻切换

## 七、平台沟通技巧

### 什么时候找渠道对接？

✅ **账户出问题**：被封、流量异常、数据异常
✅ **需要加白名单**：新功能测试、特殊权限
✅ **需要技术支持**：API对接、数据上报问题
✅ **需要行业数据**：大盘数据、竞品情况（能问到的话）
✅ **争取更好政策**：更高返点、更好的账户权重

### 沟通注意事项

1. **描述问题要清晰**
   - 账户ID：XXXX
   - 问题现象：XXXX
   - 发生时间：XXXX
   - 期望解决：XXXX

2. **态度要友好**
   - 渠道对接人也是打工人，互相体谅
   - 建立好关系，有时候能拿到内部消息

3. **问题跟进要主动**
   - 发了消息没回就再问一下
   - 重要问题直接打电话
   - 问题解决了说声谢谢

## 八、操作风险防控

### 常见操作失误

❌ **误暂停计划**：本来想点A，点成了B
❌ **出价改错了**：本来想改10块，改成100块
❌ **预算设太大**：少输了一个小数点，预算变成10倍
❌ **定向选错了**：选了完全不相关的地域

### 防错机制

1. **操作前确认**：点保存前再看一遍
2. **操作后检查**：保存后刷新确认设置正确
3. **设置预算上限**：即使出错，损失也可控
4. **双人复核**：重要操作让同事再看一眼
5. **操作日志**：记录每天做了什么调整，方便回溯

### 安全防护

- 账号密码不要泄露
- 开启二次验证
- 不要在公共电脑登录账户
- 定期修改密码

## 九、日常操作Checklist

### 每日Checklist

- [ ] 看昨日数据，消耗、转化、成本是否正常
- [ ] 检查账户余额，是否需要充值
- [ ] 检查所有计划状态是否正常
- [ ] 检查素材审核状态
- [ ] 处理超成本严重的计划
- [ ] 记录今日做了什么调整

### 每周Checklist

- [ ] 导出上周完整数据
- [ ] 做周度数据分析总结
- [ ] 整理素材库，归档表现差的素材
- [ ] 制定下周测试计划
- [ ] 和渠道对接人做一次沟通

### 每月Checklist

- [ ] 导出上月完整数据
- [ ] 做月度总结和下月规划
- [ ] 财务对账，发票处理
- [ ] 素材方向复盘，制定下月素材计划
- [ ] 团队内部复盘分享

## 十、高效操作技巧

### 技巧1：批量操作
- 重复的操作学会用批量功能
- 常用设置保存为模板，下次直接用
- 学习平台的快捷键操作

### 技巧2：报表模板化
- 把常用的报表做成固定模板
- 每次导出数据直接套用，不用重新做
- 节省大量重复劳动时间

### 技巧3：自动化工具
- 数据上报、预警等能自动化的就自动化
- 常用工具脚本化，提升效率

## 总结

**操作的核心原则：**

1. **规范**：命名、流程都要有规范，方便追溯
2. **细心**：广告操作涉及真金白银，马虎不得
3. **高效**：重复劳动想办法自动化、模板化
4. **安全**：做好风险防控，避免低级错误导致损失
